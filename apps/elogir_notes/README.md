# elogir_notes

Minimal note-taking app. Serves as a proof of concept for the elogir architecture and the [elogir_ui](../../packages/elogir_ui) design system.

Part of the [elogir monorepo](../../README.md).

## Features

- **Notes** -- create, edit, and delete notes
- **Theme toggle** -- switch between light and dark mode

## Tech Stack

- **UI**: [elogir_ui](../../packages/elogir_ui) design system
